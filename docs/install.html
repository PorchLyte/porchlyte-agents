Hire Your Team

0 of 4 complete
01
Make sure your Foundation is set up.
Every team member reads from your Voice, Brand, and Local profiles. If you haven't finished the Foundation yet, do that first. In a new chat you can confirm by checking Customize > Plugins for AI Agent Foundation. If it isn't there, set it up from the Foundation install page before you go on.
02
Add the Team marketplace.
The Team lives in its own marketplace, separate from the Foundation. Open a new chat in the Claude desktop app and add it. You only do this once.
/plugin marketplace add PorchLyte/porchlyte-agents
Tip
Run each command in its own message. Wait for Claude to finish before you send the next one.
03
Install AI Agent Team.
In the same chat, install the Team. It requires the Foundation to be installed and set up first.
/plugin install ai-agent-team@porchlyte-agents
Tip
To confirm both are active, open Customize > Plugins. You should see AI Agent Foundation and AI Agent Team listed.
04
Run /set-me-up.
Open a new chat and type the command below. This is where you meet your team and decide who to hire first. You don't have to set up all nine at once. Start with one or two that solve a real problem for you this week. Darla is the most common first hire because she pays off every morning.
/set-me-up

Connecting your tools

Some of your agents work with your real tools. Gmail, Calendar, Drive, and more. Ella, Darla, and Treena lean on Gmail; Darla also reads your Calendar; Treena and Sloane can pull from Drive. Each agent walks you through connecting what it needs the first time you run it, or you can set them up ahead of time.

To connect them now: in the Claude desktop app, click Customize in the left sidebar, then Connectors, then the + at the top. Pick a tool, click Connect, and sign in once. Any agent that needs it will have access from then on.

Getting updates

When the team is improved, you don't reinstall anything.

/plugin marketplace update porchlyte-agents

Run that in any Cowork chat and Claude pulls the latest version. Your hired team members and their personalization stay exactly as you set them up. Updates only change how the agents work.

FAQ

Do I have to hire all nine at once?+
No. /set-me-up lets you hire one, a few, or all nine. Most agents start with one or two and come back later. Your team waits for you, and your work saves between sessions.
Why does the Team need the Foundation?+
Every team member reads from your Voice, Brand, and Local profiles. Without the Foundation, the Team still works, but the writing comes out generic instead of sounding like you. Set up the Foundation first.
How do I bring an agent into a chat?+
Once hired, just say her name and what you need. "Darla, run my morning brief." "Ella, write a three-email past-client check-in." "Lia, here's an MLS sheet, give me five posts." That's it.
Can I add more team members later?+
Yes. Run /set-me-up again any time and pick up where you left off. The ones you've already hired stay ready.
What if a slash command doesn't work?+
First, confirm both plugins are installed (Customize > Plugins). Second, start a brand new chat before running /set-me-up. Third, close and reopen the desktop app. That solves it about 95% of the time.
Do I need to install on every device?+
Only on devices where you'll use Cowork. On each one, add the marketplace and install the Team. You don't have to re-run /set-me-up on every device. Your hired team lives in your Claude account.
Is my client data safe?+
With the "Help improve Claude" setting turned off (from Foundation setup), your conversations are not used for training. Treat your team the way you'd treat any tool that holds client information: don't share more than you need to, and follow your brokerage's data handling rules.
